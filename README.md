# selector
Описание класса формы в файле Form.php.

Вызов и инициализация класса происходит в файле index.php, где URL-целевой странички (for_my_frame.html) инициализируется 
в value тега <input> будущей формы. В дальнейшем пользователь может ввести любой URL.

Обрабатывая AJAX-ом onclick формы (getContent.php) получаю посредством file_get_contents файл и возвращаю его JSON на клиент в iframe.

После onload для iframe через onclick нахожу необходимые параметры элемента html-страницы, по которому был произведёт клик мыши и отправляю AJAX-ом на сервер.
Там посредством Php DOMDocument парсю его (детали в комментах файла save_to_file.php)
